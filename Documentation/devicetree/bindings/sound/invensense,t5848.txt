* Invensense T5848 MEMS Silicon Microphones

Required properties:
- compatible: Should be "invensense,t5848"
- #sound-dai-cells: should be 0.

Example of the Alif Ensemble 7 SoC:

t5848: acodec {
	#sound-dai-cells = <0>;
	compatible = "invensense,t5848";
};

Example of a "simple-audio-card":
sound-memsmic {
	compatible = "simple-audio-card";
	simple-audio-card,name = "mems-mic";
	simple-audio-card,format = "i2s";
	simple-audio-card,frame-master = <&cpudai_1>;
	simple-audio-card,bitclock-master = <&cpudai_1>;

	cpudai_1: simple-audio-card,cpu {
		sound-dai = <&i2s3>;
	};

	mcodec: simple-audio-card,codec {
		sound-dai = <&t5848>;
	};
};
